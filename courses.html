Course Summaries

SSDP-GENR

Web Programmer (Relational Databases, C# and ASP.NET Core MVC, iOS)

SSDP-GENR is the general course of the program. We start learning Relational Databases in Microsoft SQL, ASP.NET Core in C#, and iOS development in Swift. We are also introduced to other various fundamental Web Development Practices such as Angular and React, Web Security, API Development, Amazon WebServices, PHP and Laravel, Windows Forms and more. Instructors: Pat Mcgee and Phil Weier.

SSDP 1501

System Foundations 1 (Front-End HTML and CSS)

SSDP1501 focuses on front-end technologies such as HTML and CSS. We learn to make functional Web pages and how to make them beautiful. Instructor: Michael Whyte.

SSDP 2501

System Foundations 2 (JavaScript and JQuery)

SSDP2501 introduces JavaScript and JQuery and how they allow us to make powerful Web Applications. Instructor: Jason Harrison.

SSDP 3501

System Foundations 3 (Intro to PHP)

SSDP 3501 touches on PHP and server side scripting. We learn how to use the PHP tools such mySQL and Apache. Instructor: Jeff Parker.

SSDP 4001

Specialty Topics(Java and Android)

SSDP4001 focuses on Java development and how it's similar to C# but also different in so many ways. We are also introduced to Android Development using Android Studio. Instructor: Paul Mills.

SSDP 5001

MEAN stack (Node.js and other tools)

SSDP5001 is all about using Node.js and all the powerful tools it provides. We learn to develop in the MEAN (MongoDB - ExpressJS - Angular - Node.js) stack. Instructor: Medhat Elmasry
